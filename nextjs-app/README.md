# Fire Department Radio Transcription System - Next.js Application

## Overview

This is the modern Next.js reimplementation of the Fire Department Radio Transcription & Compliance Audit System. It replaces the Streamlit prototype with an enterprise-grade, accessible, and emergency-optimized web application.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **UI Components**: Shadcn/ui (Radix UI primitives)
- **Documentation**: Storybook 8
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Quick Start

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run Storybook
npm run storybook

# Build for production
npm run build

# Start production server
npm start
```

### Development URLs

- **Next.js App**: http://localhost:3000
- **Storybook**: http://localhost:6006

## Design System

The application uses a comprehensive fire department-specific design system. See [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) for complete documentation.

### Key Features

1. Mission-Critical Speed - Information accessible within seconds
2. Stress-Reduced Interface - Clear visual hierarchy under pressure
3. Emergency-Ready Visuals - High contrast for outdoor visibility
4. Touch-Friendly - Minimum 44px targets for gloved hands
5. WCAG 2.2 AA Compliant - Full accessibility
